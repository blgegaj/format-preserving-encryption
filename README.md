
# Installation

```shell
git clone https://github.com/blgegaj/format-preserving-encryption.git
cd format-preserving-encryption/
pip install -r requirements.txt
```

## Example

```shell
python src.py

Number to encrypt - 7777777777777777
Encrypted number - 6584286078918194

Decrypting ...

Ciphertext to decrypt - 6584286078918194
Decrypted number - 7777777777777777

```
